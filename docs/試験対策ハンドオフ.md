## このアプリは何？
鍼灸国家試験向けに作ったブラウザ学習アプリですが、**中身は科目非依存**。
どんな試験の問題・教材でも取り込めます。データは端末（ブラウザ）に保存。
公開URL: https://youtaix656-eng.github.io/-/

## 筆記テストに使える機能
- **一問一答**（四択・○×）… 基本の演習
- **音声学習**（ハンズフリー）＋**連結学習モード1〜10**
  （連鎖／比較・対比／数値まとめ／穴埋め／弱点優先／選択肢読み上げ／章通し読み／
  つながりナレーション／適応／今日の連結）… 移動中・家事中の“ながら学習”
- **復習（SRS）** … 間違いを間隔反復で定着
- **模試** … 時間制限・合格ライン判定つきの通し演習
- **ダッシュボード** … 弱点（正答率の低い所）の可視化
- **マインドマップ** … まぎらわしい比較・数値・50音索引
- **自由文/ノートから自動問題化** … 教材テキストを貼ると問題化
- **カレンダー** … 試験日までの学習計画

## 実技テストに使える機能（手順・順序の暗記）
- **穴埋め音声** … 手順書の要語を伏せて出題→答え。順序を耳で覚える
- **比較・対比読み** … 「やりがちな誤り ⇔ 正しいやり方」を対で
- **メモ** … 安全確認・手順のチェックリスト
- **一問一答化** … 「次の手順は？」「この時のNGは？」を四択/○×に

## ★ 取り込み形式（この形で作れば、そのままアプリに入る）
### CSV（1行目はヘッダー）
```
科目,問題文,選択肢,正解,解説
安全衛生,消火器の使用でまず行うのは？,安全ピンを抜く|ホースを向ける|レバーを握る|周囲に知らせる,1,火事に気づいたらまず周囲に知らせ、安全ピンを抜く。
接客,クレーム対応で最初にすべきは？,言い訳する|傾聴して謝意を示す|上司を呼ぶ|反論する,2,まず相手の話を傾聴し、不快にさせたことへの謝意を示す。
```
- 選択肢は「|」区切り。**選択肢を空にすると○×問題**（正解は ○/× で書く）。
- 正解は **1始まりの番号** か、選択肢の文字列そのもの。

### JSON配列（どちらでも可）
```json
[
  {"subject":"安全衛生","type":"choice","question":"…",
   "choices":["…","…","…","…"],"answer":0,"explanation":"…","tags":["消火器","初期対応"]}
]
```
- JSON の `answer` は **0始まりのインデックス**。`type` は "choice" か "ox"。
- `tags`（重要語）を付けると、音声の連結モードとマインドマップで活きます。

## ★ 問題を作らせる汎用プロンプト（他AIへ）
```
次の【教材/過去問】から学習問題を作ってください。
・原問（4択）×1：内容を忠実に。解説は要点を自分の言葉で。
・一問一答 ×4：同じ論点を別角度で、答えが互いに重複しないように
  （A 核心 / B 定義・逆引き / C 鑑別＝紛らわしい点 / D ○×）。
・各問に tags（重要語 2〜4個、正式名称）。
・専門内容は正確に。曖昧・古い数字は「※要確認」を付す。
・被り厳禁・網羅的に（元の全選択肢と要点をどこかでカバー）。
出力は次のCSVのみ：科目,問題文,選択肢(|区切り),正解(1始まり),解説
【教材/過去問】↓
（ここに貼る）
```

## アプリへの入れ方
1. 上のプロンプトで作った **CSV/JSON を私（Claude Code）に渡す** →
   端末だけに取り込む「1タップ取り込みリンク」にして返します（推奨・確実）。
2. または各機能の取り込み/自由文入力から直接読み込む。
- 取り込み後は一問一答・音声・復習・模試・マインドマップで使えます。
- 同じ問題は自動で重複除外されます。
